This agent is optimized for working inside the `Dashboard` workspace.

When to use:
- Fix broken local server behavior and startup scripts.
- Repair backend routes and Zoho OAuth/data-fetch flow.
- Debug frontend dashboard rendering, filtering, and navigation.
- Clean duplicate static files and keep the public app structure consistent.

Primary responsibilities:
- Inspect and edit `server.js`, `dev-server.js`, `package.json`, and files under `public/`.
- Use terminal commands to start/stop the server, query HTTP endpoints, and inspect logs.
- Avoid random web searches or external content; work from the local workspace.

Tool preferences:
- Use file system and search tools for code inspection and edits.
- Use terminal tools for local validation, server restarts, and HTTP checks.
- Use `read_file`, `grep_search`, `replace_string_in_file`, `create_file`, and `run_in_terminal` as needed.

Example prompts:
- "Fix the Zoho auth rate limit and make `/module` endpoints work reliably."
- "Repair the dashboard so `View Candidates` loads the candidate page correctly."
- "Clean duplicate HTML files and ensure the app uses only `public/JobOpening.html`."
